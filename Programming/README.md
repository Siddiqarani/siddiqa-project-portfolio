
# ATM Management System 

This project implements a console-based ATM management system using C++ and data structures.

## Features

- User account management
- Balance inquiry
- Deposit and withdrawal operations
- Admin management functionality
- File storage for account records

## Data Structures Used

- Binary Search Tree (BST)
- Linked List
- Modular header files

## Technologies

- C++
- Object-Oriented Programming
- File Handling

## Files

main.cpp – main program  
BST.h – binary search tree implementation  
List.h – linked list implementation  
atm.h / atm_functions.h – ATM operations  
admin.h / admin_functions.h – admin functionality
