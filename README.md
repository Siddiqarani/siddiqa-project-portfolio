## Bioinformatics Projects

### DNA Sequence Analyzer (Python)
A Python-based bioinformatics tool for analyzing DNA sequences.

Features:
- GC content calculation
- Reverse complement generation
- DNA transcription
- Protein translation

Tools: Python, BioPython, Matplotlib

### Bisulfite Sequencing DNA Methylation Pipeline
A simulated bisulfite sequencing workflow for DNA methylation analysis.

Workflow:
- BS-seq read simulation using Sherman
- Quality trimming with Trim Galore
- Alignment using Bismark and Bowtie2
- Extraction of CpG, CHG, and CHH methylation calls

Tools: Sherman, Trim Galore, Bismark, Bowtie2

### Molecular Docking of Favipiravir with SARS-CoV-2 RdRp
Performed protein–ligand docking to study antiviral drug interactions.

Tools:
- AutoDock Vina
- UCSF Chimera

Objective:
To analyze binding affinity and molecular interactions between Favipiravir and SARS-CoV-2 RdRp.
