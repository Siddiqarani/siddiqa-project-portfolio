# DNA Sequence Analyzer

## Description
This project analyzes DNA sequences and performs basic bioinformatics operations such as GC content calculation, reverse complement generation, transcription, and translation.

## Tools Used
Python
BioPython
Matplotlib

## Features
- GC content calculation
- Reverse complement
- DNA → RNA transcription
- RNA → protein translation
- Sequence visualization

## Input
FASTA file containing DNA sequence.

## Output
GC content statistics and visualizations.
